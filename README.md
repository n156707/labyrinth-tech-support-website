# Labyrinth Tech Support Website

This repository contains the static files for the Labyrinth Tech Support web
site. The site includes an optional high‑contrast mode optimised for users with
low vision.

## Accessibility

Use the **Accessibility** toggle button in the page header (or press <kbd>Alt</kbd>
 + <kbd>A</kbd>) to switch to high‑contrast mode. The `high-contrast.css` file
defines a colour palette of black backgrounds, white text and bright yellow and
cyan accents to maximise readability.
